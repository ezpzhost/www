API

Documentation for the API is available at api.ezpz.host/docs

Authentication

All API queries must be made with your self-generated secret token in the Authorization header. See token for details.

curl -X POST https://api.ezpz.host/billing/fund \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"size":"xs","days":30}'
