VPS

Creating, rebuilding, and deleting a server are all asynchronous, the request returns immediately, and you poll status until it's done.

Status

A VPS moves through these states:

StatusMeaning
provisioningBeing created
activeRunning and reachable
rebuildingBeing reinstalled in place
deletingBeing destroyed
deletedGone
errorSomething went wrong; not billed while in this state, cleaned up automatically if it stays broken

Rebuilding reinstalls the server from the same image you originally specified, your IP address stays the same.

Endpoints

MethodPathDescription
GET/vps/statusList your VPSs, or one with ?id=
POST/vps/createCreate a VPS (needs a funded day of the requested size)
POST/vps/deleteDelete a VPS
POST/vps/rebuildReinstall in place, same IP

Example

curl -X POST https://api.ezpz.host/vps/create \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "region": "usa",
    "size": "xs",
    "image": "ubuntu",
    "ssh_key": "ssh-ed25519 AAAA..."
  }'
